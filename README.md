### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息 
5. 汽车投放
6. 使用订单管理
7. 汽车归还管理


**管理员功能：**
1. 登录
2. 个人中心
3. 用户管理
4. 投放地区管理
5. 汽车信息管理
6. 汽车投放管理
7. 汽车入库管理
8. 使用订单管理
9. 汽车归还管理
10. 管理员管理
